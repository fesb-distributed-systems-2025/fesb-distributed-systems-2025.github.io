Hints​

  • Do not forget to enable CORS on backend.
    • For more information refer to this COMMIT in pkaselj-lab-07 repo
  • Do not forget to refer to this COMMIT in pkaselj-lab-08-simple repo to see how to adjust given frontend for your backend.
  • Install Live Server extension in Visual Studio Code to serve your frontend page locally (more information below)
  • Do not forget that the frontend must send objects that your controller expects i.e. your DTOs

General​

  • Refer to the pkaselj-lab-07 GitHub Repo for reference implementation.
  • Refer to the pkaselj-lab-08-simple GitHub Repo for reference implementation of basic frontend in HTML and CSS.
  • Refer to the pkaselj-lab-08 GitHub Repo for reference implementation of a frontend web page using Bootstrap.
  • Refer to the pkaselj-lab-07-plus GitHub Repo for last year's full project implementation.
  • Use regexr to practise regular expressions

Installing Live Server Extension in Visual Studio Code​

  1. Open Extensions tab in Visual Studio Code as in figure below

  1. Search for, and install, Live Server extension

Start Hosting Your Web Page in Visual Studio Code​

  1. In the lower right corner of the VS Code, press the button Go Live

  1. Alternatively, go to the tool bar in the upper right corner, View > Command Palette... > "> Live Server: Open with Live Server"
